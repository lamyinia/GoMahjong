# 麻将中的“筋牌”详解

筋牌是日本麻将中**防守理论的核心概念**，用于推测对手不要的牌，从而降低放铳风险。让我详细解释这个重要的概念。

## 一、什么是筋牌？

### 1. 基本定义

**筋牌**指数牌（万、筒、索）中**数字相差3**的牌。它们之间存在特殊的听牌关联性。

### 2. 三条“筋”

数牌1-9分为三条筋：

| 筋名        | 包含牌  | 关系说明 |
| ----------- | ------- | -------- |
| **1-4-7筋** | 1、4、7 | 间隔3    |
| **2-5-8筋** | 2、5、8 | 间隔3    |
| **3-6-9筋** | 3、6、9 | 间隔3    |

**记忆口诀**：1-4-7，2-5-8，3-6-9，各成一筋。

## 二、筋牌的原理

### 为什么筋牌安全？

当对手听**两面听牌**时，他需要的牌通常是**同一筋上的相邻牌**。

**示例**：

```
对手手牌有：4万 5万
听牌：3万和6万（两面听）

3万属于1-4-7筋
6万属于3-6-9筋
```



- 如果对手**不要**某张筋牌，他很可能**也不要**同一筋上的其他牌

- 但要注意：这**不是绝对的**，只是概率上的安全

## 三、筋牌的分类

### 1. 完全筋（フルスジ）

同一筋上的三张牌都安全。

**示例**：

- 1-4-7筋：如果对手打过1万，那么4万、7万相对安全

- 2-5-8筋：如果对手打过5筒，那么2筒、8筒相对安全

### 2. 半筋（ハンスジ）

只有部分筋牌被确认安全。

**示例**：

- 打过1万 → 4万相对安全（但7万不一定）

- 打过4万 → 1万和7万相对安全

### 3. 壁筋（カベスジ）

因牌山枯竭而形成的特殊筋。

## 四、筋牌在防守中的应用

### 1. 读牌基础

```
对手打过：
- 1万（1-4-7筋）
- 5筒（2-5-8筋）

相对安全的牌：
- 1-4-7筋：4万、7万
- 2-5-8筋：2筒、8筒
```

### 2. 立直防守

当对手立直时，筋牌是**重要的安全牌参考**：

| 对手打过 | 相对安全的筋牌 | 安全等级 |
| -------- | -------------- | -------- |
| 2万      | 5万、8万       | 较高     |
| 5索      | 2索、8索       | 较高     |
| 4筒      | 1筒、7筒       | 较高     |

### 3. 实战示例

```
情况：南家立直，牌河中他打过5万
您的手牌有危险张：2万、8万

分析：
5万属于2-5-8筋
2万和8万是同一筋的牌
根据筋牌理论，2万和8万相对安全
但仍需结合其他信息判断
```

## 五、筋牌的局限性

### 1. 筋牌不是绝对安全

筋牌安全是基于**对手是两面听牌**的假设。但对手可能是：

| 听牌类型 | 筋牌安全性 |
| -------- | ---------- |
| 两面听   | 相对安全   |
| 坎张听   | 不完全安全 |
| 边张听   | 不完全安全 |
| 单骑听   | 不安全     |
| 多面听   | 不安全     |

### 2. 特殊形状破坏筋牌理论

**示例1：愚形听牌**

```
手牌：3万 5万
听牌：4万（坎张）
这时筋牌理论不适用
```

**示例2：包含筋牌的复杂听牌**

```
手牌：1万 2万 3万 4万
听牌：1万、4万（双碰）
这时1万和4万都是危险牌
```

## 六、扩展概念

### 1. 壁（カベ）

数字4和6称为5的“壁”，因为：

- 4是5的下壁

- 6是5的上壁

**应用**：如果4和6都现物（被安全打出），那么5相对安全。

### 2. 间4间（あいだよんあいだ）

指间隔4的牌，如1-5-9，也有一定的参考价值。

### 3. 筋牌与现物的区别

- **现物**：绝对安全（对手舍牌中已有）

- **筋牌**：相对安全（基于概率推断）

## 七、雀魂中的筋牌提示

在雀魂游戏中，筋牌通常有视觉提示：

### 1. 舍牌显示

- 对手打过的牌会显示在牌河中

- 可以手动标记危险牌

### 2. 防守辅助

有些插件或模式会显示：

- 筋牌高亮

- 安全度评级

- 推荐切牌

### 3. 实战界面

```
牌河显示对手打过：3筒、6筒
系统提示：3-6-9筋相对安全
```

## 八、防守策略结合

### 1. 综合判断要素

| 要素 | 说明               |
| ---- | ------------------ |
| 筋牌 | 基本参考           |
| 现物 | 绝对安全           |
| 牌河 | 观察对手打牌习惯   |
| 场况 | 点数、巡目、副露等 |
| 手牌 | 自己的听牌情况     |

### 2. 防守优先级

```
安全度排序：
1. 现物（对手打过）> 
2. 全筋牌（三张都出现）> 
3. 半筋牌 > 
4. 壁牌 > 
5. 其他
```

## 九、筋牌记忆技巧

### 1. 数字规律

```
1-4-7：+3 +3
2-5-8：+3 +3  
3-6-9：+3 +3
```

### 2. 九牌分组

```
将1-9分成三组：
[1,4,7]、[2,5,8]、[3,6,9]
每组内牌相互关联
```

### 3. 实战记忆法

记住常见组合：

- 中张筋：4-7、5-8、6-9

- 边张筋：1-4、2-5、3-6

- 延伸筋：1-4-7全线

## 十、高级筋牌理论

### 1. 双重筋

当两张筋牌都出现时，安全性更高。

**示例**：

```
对手打过：2万和5万
安全牌：8万（2-5-8筋全现）
```

### 2. 筋牌与牌效率

进攻时也要考虑筋牌：

- 保留筋牌搭子效率高

- 筋牌容易形成两面搭

### 3. 心理战

有时故意打筋牌诱导对手：

- 早期打筋牌显示“不要”

- 实际听筋牌上的其他牌

## 十一、开发中的应用

如果您开发麻将AI，筋牌理论可用于：

### 1. AI防守逻辑

```
type DefenseAI struct {
    // 记录对手舍牌
    discardedTiles map[int][]Tile
    
    // 计算安全度
    CalculateSafety(tile Tile, playerIndex int) float64 {
        safety := 0.0
        
        // 检查是否现物
        if isGenbutsu(tile, playerIndex) {
            safety += 100.0
        }
        
        // 检查筋牌
        if isSuji(tile, playerIndex) {
            safety += 70.0
        }
        
        // 检查壁
        if isKabe(tile, playerIndex) {
            safety += 50.0
        }
        
        return safety
    }
}
```

## 十二、常见问题

### Q1：筋牌100%安全吗？

**不**。筋牌是概率安全，不是绝对安全。高手有时会故意听筋牌。

### Q2：什么时候筋牌理论失效？

- 对手愚形听牌

- 对手多面听

- 对手故意欺骗

- 终盘特殊形势

### Q3：新手如何运用筋牌？

1. 先记住三条筋：1-4-7，2-5-8，3-6-9

2. 观察对手打牌，记录筋牌

3. 防守时优先打现物，其次打筋牌

4. 结合场况判断

## 总结

**筋牌是麻将防守的重要工具**，但需理性看待：

### 核心要点

1. **定义**：数字相差3的牌（1-4-7，2-5-8，3-6-9）

2. **原理**：基于对手两面听牌的假设

3. **作用**：推测相对安全的牌

4. **局限**：不是绝对安全，需结合其他信息

### 使用建议

- **进攻时**：利用筋牌原理做高效率手牌

- **防守时**：筋牌是重要参考，但不是唯一依据

- **学习时**：从记住三条筋开始，逐步深入

在雀魂等游戏中，掌握筋牌理论能显著提升防守水平，减少放铳。但切记麻将没有绝对，灵活运用才是关键。
