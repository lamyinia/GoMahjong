# 庄家（親・オヤ）详解

庄家是立直麻将中**最关键的角色**，直接影响游戏策略、得分计算和游戏进程。以下是庄家的全面解释：

## 一、庄家的基本概念

### 定义

- 

  **庄家**：当前局中坐**东风位**的玩家

- 

  **闲家**：其他三位玩家（子・コ）

- 

  每局有且只有一位庄家

### 标识

- 

  场风表示牌前的**红色标识**

- 

  雀魂中：庄家座位有"东"字标识，出牌时牌背有红点

## 二、庄家的特权

### 1. 和牌得分更高

| 和牌类型 | 闲家和牌    | 庄家和牌      |
| -------- | ----------- | ------------- |
| 自摸     | 闲家各付1份 | 闲家各付1.5份 |
| 荣和     | 放铳者付1份 | 放铳者付1.5份 |

**举例**：满贯（8000点）

- 

  闲家自摸：2000/4000（闲家付2000，庄家付4000）

- 

  庄家自摸：4000每家（闲家各付4000）

- 

  相差**一倍**

### 2. 连庄权利

- 

  庄家和牌 → 继续坐庄

- 

  流局时庄家听牌 → 继续坐庄

- 

  流局时庄家不听牌 → 下庄

### 3. 起家优势

- 

  开局多摸一张牌（14张 vs 闲家13张）

- 

  第一个出牌

- 

  更容易做牌

## 三、庄家的责任

### 1. 支付更多

- 

  支付闲家自摸时需付1.5倍

- 

  被闲家荣和时支付1.5倍

### 2. 被追加倍

- 

  庄家放铳时，惩罚加倍

- 

  尤其要避免被闲家大牌直击

### 3. 控制游戏节奏

- 

  连庄会增加"本场棒"，每局+300分

- 

  庄家强攻会加速游戏进程

## 四、庄家轮转规则

### 1. 基本轮庄顺序

```
东1局：玩家A（起家）为庄家
    ↓ 庄家未和牌且流局不听 → 下庄
东2局：玩家B为庄家
    ↓
东3局：玩家C为庄家
    ↓
东4局：玩家D为庄家
    ↓
南1局：玩家A为庄家
    ↓
南2局：玩家B为庄家
    ↓
南3局：玩家C为庄家
    ↓
南4局：玩家D为庄家
```

### 2. 连庄情况

```
东1局：玩家A（庄家）和牌
    ↓ 连庄，本场+1
东1局1本场：玩家A继续庄家
    ↓ 再次和牌
东1局2本场：玩家A继续庄家
    ↓ 闲家和牌
东2局：玩家B成为庄家（本场重置为0）
```

## 五、庄家策略特点

### 进攻策略

1. 

   **积极做牌**：争取和牌连庄

2. 

   **立直果断**：立直后追立直棒

3. 

   **做大牌倾向**：利用倍数优势

### 防守策略

1. 

   **避免放铳**：尤其避开车、跳满以上大牌

2. 

   **早期防守**：感觉形势不利时早守

3. 

   **控制手牌**：保持听牌可能以争取流局连庄

## 六、本场和立直棒

### 本场（ボーナス）

- 

  庄家连庄一次，本场数+1

- 

  每1本场 = 和牌时额外+300分

- 

  累积无上限

### 立直棒

- 

  立直时支付的1000点

- 

  放置在桌中央

- 

  下次和牌者获得所有立直棒

- 

  庄家立直棒价值更高（易被争夺）

## 七、得分计算差异

### 符数计算差异

| 项目     | 闲家 | 庄家 |
| -------- | ---- | ---- |
| 自摸底符 | 20符 | 20符 |
| 自摸符   | +2符 | +0符 |
| 荣和底符 | 30符 | 30符 |

### 基本点计算

```
基本点 = 符 × 2^(番+2)
闲家得分 = 基本点 × 倍数
庄家得分 = 基本点 × 倍数 × 1.5
```

### 具体得分对照

| 番数    | 闲家自摸  | 庄家自摸 | 闲家荣和 | 庄家荣和 |
| ------- | --------- | -------- | -------- | -------- |
| 1番30符 | 500/1000  | 1500     | 1000     | 1500     |
| 3番40符 | 1300/2600 | 3900     | 5200     | 7700     |
| 满贯    | 2000/4000 | 6000     | 8000     | 12000    |

## 八、流局处理

### 四种流局结果

1. 

   **庄家听牌，闲家有听**：庄家连庄，本场+1

2. 

   **庄家听牌，闲家全不听**：庄家连庄，本场+1，支付不听罚符

3. 

   **庄家不听，闲家有听**：庄家下庄，支付不听罚符

4. 

   **全不听牌**：庄家下庄，本场重置

### 不听罚符

- 

  不听者支付听牌者

- 

  庄家不听支付闲家的1.5倍

- 

  通常：不听者支付1000点（庄家付1500）

## 九、雀魂中的庄家实现

### 视觉标识

1. 

   **座位标识**：庄家位置显示"东"字

2. 

   **牌背标记**：庄家手牌有红点标识

3. 

   **UI强调**：庄家分数框有特殊边框

4. 

   **动画效果**：庄家和牌有特殊动画

### 规则调整

1. 

   **西入取消**：南4局庄家无论输赢都下庄

2. 

   **时间压力**：庄家出牌时间相同，但心理压力更大

3. 

   **匹配平衡**：庄家优势在匹配算法中有所考虑

### 统计数据显示

- 

  庄家平均和牌率：约28-32%（闲家约22-25%）

- 

  庄家平均得点：+1500～+2000点/局

- 

  连庄率：约35-40%的局有连庄

## 十、对游戏进程的影响

### 游戏节奏

```
庄家强势 → 连庄多 → 本场累积 → 得分差拉大 → 游戏可能提前结束
庄家弱势 → 轮庄快 → 本场少 → 得分接近 → 游戏进行到南场
```

### 心理影响

1. 

   **庄家心理**：

   - 

     压力大，但机会多

   - 

     倾向进攻，不怕防守

   - 

     希望尽早和牌确立优势

2. 

   **闲家心理**：

   - 

     对庄家更谨慎

   - 

     更愿防守避铳

   - 

     瞄准庄家做大牌

## 十一、特殊规则情况

### 东风战

- 

  只有东风场4局

- 

  庄家价值更高（每局占比大）

- 

  连庄影响更大

### 三人麻将

- 

  庄家轮转加快

- 

  庄家出现频率更高

- 

  策略调整明显

### 古役规则

- 

  某些古役规则中庄家特权更大

- 

  如"两翻缚"等限制闲家

## 十二、开发注意事项

### 状态管理

```
type GameState struct {
    DealerIndex   int  // 庄家座位(0-3)
    Honba         int  // 本场数
    RoundWind     Wind // 场风(东/南)
    RoundNumber   int  // 局数(1-4)
    RiichiSticks  int  // 立直棒数量
    IsDealerTurn  bool // 当前是否庄家回合
}
```

### 得分计算

```
func CalculatePayment(basePoints int, isDealerWin bool, isDealerPay bool) map[int]int {
    payments := make(map[int]int)
    
    if isDealerWin {
        // 庄家和牌
        payment = basePoints * 2 * 1.5
    } else if isDealerPay {
        // 庄家支付闲家
        payment = basePoints * 1.5
    } else {
        // 闲家之间支付
        payment = basePoints
    }
    
    return payments
}
```

### 轮庄逻辑

```
func NextDealer(currentDealer int, isDealerWin bool, isDealerTenpai bool) (int, int) {
    if isDealerWin || isDealerTenpai {
        // 连庄
        return currentDealer, 1
    } else {
        // 轮庄
        newDealer := (currentDealer + 1) % 4
        return newDealer, 0
    }
}
```

## 总结

庄家是立直麻将的**核心机制**，创造了游戏的：

1. 

   **动态平衡**：庄闲优劣势交替

2. 

   **战略深度**：不同身份的攻守选择

3. 

   **节奏控制**：连庄加速，轮庄缓和

4. 

   **逆转可能**：庄家一波流可大幅逆转

在雀魂等现代游戏中，庄家规则被**完整保留**，因为这是立直麻将的**灵魂所在**。虽然取消了西入控制时长，但庄家的核心特权全部保留，确保了游戏的竞技性和策略深度。

开发时应特别注意庄家规则的正确实现，这直接关系到游戏体验的**正统性**和**竞技平衡**。
